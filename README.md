# probot-merge-when-green

> A GitHub App built with [Probot](https://github.com/probot/probot) that Automatically merge pull requests when all CI statuses succeed

## Setup

```sh
# Install dependencies
npm install

# Run typescript
npm run build

# Run the bot
npm start
```
